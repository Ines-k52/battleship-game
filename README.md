This is a simple console-based implementation of the classic Battleship game written in C++. The game is played on a 4×4 grid where a set of ships are hidden at predefined locations.

The player attempts to locate and “hit” all ships by entering row and column coordinates. After each guess, the program provides immediate feedback indicating whether the attempt was a hit or a miss. When a ship is hit, it is removed from the grid to prevent duplicate scoring.

The game continues until all ships are found. At the end, the total number of turns taken to achieve victory is displayed.
